# Biryani vs Pulao Classifier üçõ

A basic deep learning web app that classifies rice dishes as **Biryani** or **Pulao**.

---

This biryani-pulao classifier has been built using **TensorFlow** and trained through **transfer learning** with **MobileNetV2**.  
The web app was made using **Streamlit**.

Upload a picture of a rice dish, and it predicts if it's **Biryani** or **Pulao** with a confidence metric.
